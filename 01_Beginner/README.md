# 01_Beginner: The Fundamentals of Go

This module covers the absolute basics of Go. Here you will learn how to write basic syntax, manage variables, structure conditional logic, and handle files.

## Modules

* **`01_hello_world`**: The entry point to Go, understanding packages and imports.
* **`02_variables_and_types`**: Declaring strong-typed variables (`int`, `string`, `bool`) and type inference.
* **`03_user_input`**: Reading from `os.Stdin` via `bufio` and `fmt.Scan`.
* **`04_math_and_crypto`**: Basic arithmetic operations and using the `math` and `crypto` packages.
* **`05_pointers`**: Introduction to memory addresses and passing by reference versus passing by value.
* **`06_arrays_and_slices`**: Fixed-size arrays vs dynamically-sized slices, and `make`.
* **`07_maps`**: Key-value pair hash maps for O(1) lookups.
* **`08_structs`**: Creating custom, composite data types.
* **`09_if_else`**: Flow control and initialization statements inside `if` blocks.
* **`10_switch`**: Advanced switch statements without implicit fallthrough.
* **`11_loops`**: Go's only looping construct (`for`), acting as `while`, `do-while`, and range iterators.
* **`12_functions`**: Creating reusable functions, multiple return values, and anonymous functions.
* **`13_methods`**: Attaching functions to specific struct types (value vs pointer receivers).
* **`14_defer`**: Scheduling function execution right before the parent function returns (e.g., closing files).
* **`15_files`**: Reading and writing bytes to the local filesystem natively.
